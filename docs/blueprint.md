# **App Name**: TimeFlow

## Core Features:

- User Authentication: Secure login page with email/password, password view/hide, and 'Continue with Google/Microsoft' options.
- Calendar View: Display a calendar with options to view events by day, week, or month.
- Event Creation: Create and schedule events with date, time, and description. Enable modification later.
- Event Listing: List past and future events in a chronological order.
- Reminders: Set up reminders for upcoming events with customizable alerts.
- Sidebar Navigation: Quick access to key features and calendar views through a responsive sidebar.
- AI-Powered Smart Scheduling: Suggest optimal times for events based on user's existing schedule and preferences. The LLM acts as a tool in making time suggestions based on patterns from the user's current calendar, when available.

## Style Guidelines:

- Primary color: Soft Lavender (#D0BFFF) for a calming and organized feel.
- Background color: Light Gray (#F5F5F5), offering a clean backdrop to focus on content.
- Accent color: Muted Purple (#947CB0), providing highlights on interactive elements and important dates.
- Body and headline font: 'PT Sans', a humanist sans-serif, should convey both modernity and warmth. It is appropriate for both headlines and body text.
- Use minimalist icons for calendar functions, event types, and sidebar actions.
- Employ a clean, card-based layout for event listings, with clear delineation of date, time, and details.
- Subtle animations for transitions between calendar views and loading event details.
